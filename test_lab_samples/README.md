# Test Lab Sample Files

These files are lightweight sample inputs for the UI Test Lab.

Use them to verify:

- Test A: upload `transcripts/*.json` and optionally `verdicts.json`
- Test B: upload `prompt-test.md` or leave prompt blank, then upload `transcripts/*.json`
- Test C: upload `prompt-test.md` as the current prompt, `prompt-baseline.md` as the baseline, and `transcripts/*.json`

Included:

- `transcripts/` - 3 sample call JSON files
- `verdicts.json` - expected verdicts for the sample transcripts
- `prompt-test.md` - stronger test prompt
- `prompt-baseline.md` - weaker baseline prompt

These files are designed for UI testing and demo purposes, not for the main assignment scoring.
